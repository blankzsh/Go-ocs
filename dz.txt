OCS网课助手AI题库系统使用教程

[align=center][color=red][size=5][b]一、项目简介[/b][/size][/color][/align]

OCS网课助手AI题库系统是专为在线课程学习者设计的智能答题工具。它能够自动从多个AI平台获取题目答案，并将结果缓存到本地数据库中，帮助学生在网课学习过程中快速解决题目问题，提高学习效率。

[align=center][color=red][size=5][b]二、功能特性[/b][/size][/color][/align]

[list]
[*][b]多AI平台支持：[/b]
  硅基流动(SiliconFlow)、阿里云百炼、智普AI、Ollama（本地模型）、DeepSeek、ChatGPT、Gemini

[*][b]双数据库支持：[/b]
  MySQL（适用于生产环境）、SQLite（适用于轻量级部署）

[*][b]智能缓存机制：[/b]
  自动缓存已查询的题目答案，减少重复API调用，提高响应速度

[*][b]RESTful API接口：[/b]
  提供标准API供其他应用调用

[*][b]管理后台：[/b]
  Web界面查看统计信息，浏览和搜索已缓存的题目答案
[/list]

[align=center][color=red][size=5][b]三、系统要求[/b][/size][/color][/align]

[list]
[*][b]操作系统：[/b]Windows 7及以上 / Linux / macOS
[*][b]Go版本：[/b]1.21或更高版本
[*][b]数据库（任选其一）：[/b]
  MySQL 5.7或更高版本、SQLite 3（内置支持，无需额外安装）
[/list]

[align=center][color=red][size=5][b]四、安装步骤[/b][/size][/color][/align]

[b]1. 安装Go语言环境：[/b]
[list]
[*]访问 [url=https://golang.org/dl/]https://golang.org/dl/[/url] 下载适合您操作系统的Go安装包
[*]按照官方指南完成安装
[*]验证安装：打开命令行执行 [code]go version[/code]
[/list]

[b]2. 获取项目源码：[/b]
[list]
[*]方法一：使用Git克隆
[code]git clone https://github.com/blankzsh/Go-ocs.git
cd Go-ocs[/code]

[*]方法二：直接下载ZIP包并解压
[/list]

[b]3. 安装依赖：[/b]
打开命令行，进入项目根目录，执行：
[code]go mod tidy[/code]

[b]4. 配置系统：[/b]
[list]
[*]复制配置文件：
[code]cp configs/config.example.json configs/config.json[/code] (Linux/macOS)
[code]copy configs\config.example.json configs\config.json[/code] (Windows)

[*]编辑 [i]configs/config.json[/i] 文件：
[code]{
  "host": "127.0.0.1",
  "port": 8000,
  "platform": "siliconflow",
  "database_type": "mysql",
  "api_keys": {
    "aliyun": "your_aliyun_api_key_here",
    "siliconflow": "your_siliconflow_api_key_here",
    "zhipu": "your_zhipu_api_key_here",
    "deepseek": "your_deepseek_api_key_here",
    "chatgpt": "your_chatgpt_api_key_here",
    "gemini": "your_gemini_api_key_here"
  },
  "models": {
    "aliyun": "qwen-plus-latest",
    "siliconflow": "deepseek-ai/DeepSeek-R1",
    "zhipu": "glm-4",
    "ollama": "llama3",
    "deepseek": "deepseek-chat",
    "chatgpt": "gpt-3.5-turbo",
    "gemini": "gemini-pro"
  },
  "mysql": {
    "host": "127.0.0.1",
    "port": 3306,
    "user": "root",
    "password": "your_mysql_password",
    "database": "question_bank"
  },
  "sqlite": {
    "path": "question_bank.db"
  }
}[/code]

[*]配置说明：
  [list]
  [*][b]host:[/b] 服务器监听地址
  [*][b]port:[/b] 服务器监听端口
  [*][b]platform:[/b] 使用的AI平台（siliconflow/aliyun/zhipu/ollama/deepseek/chatgpt/gemini）
  [*][b]database_type:[/b] 数据库类型（mysql/sqlite）
  [*][b]api_keys:[/b] 各平台的API密钥
  [*][b]models:[/b] 各平台使用的模型
  [*][b]mysql:[/b] MySQL数据库配置
  [*][b]sqlite:[/b] SQLite数据库配置
  [/list]
[/list]

[align=center][color=red][size=5][b]五、运行系统[/b][/size][/color][/align]

[b]1. 启动服务：[/b]
在项目根目录下执行：
[code]go run cmd/main.go[/code]

[b]2. 确认服务启动成功：[/b]
[list]
[*]查看控制台输出，确认无错误信息
[*]记住显示的API配置信息，后续OCS网课助手配置会用到
[/list]

[b]3. 访问管理后台：[/b]
打开浏览器访问：[url=http://127.0.0.1:8000/admin/]http://127.0.0.1:8000/admin/[/url]
可以查看系统统计信息、浏览和搜索已缓存的题目答案

[align=center][color=red][size=5][b]六、配置工具使用[/b][/size][/color][/align]

为了方便配置系统，项目提供了命令行配置工具：

[b]1. Windows系统：[/b]
[list]
[*]英文版：[code]powershell -ExecutionPolicy Bypass -File .\configurator.ps1[/code]
[*]中文版：[code]powershell -ExecutionPolicy Bypass -File .\configurator_zh.ps1[/code]
[/list]

[b]2. Linux/Mac系统：[/b]
[code]./configurator.sh[/code]

通过配置工具可以：
[list]
[*]选择AI平台
[*]配置各平台API密钥
[*]设置各平台使用的模型
[*]选择数据库类型
[*]配置数据库连接参数
[/list]

[align=center][color=red][size=5][b]七、API使用方法[/b][/size][/color][/align]

[b]1. 查询题目答案：[/b]
[code]GET /api/query?title=问题内容[&options=选项内容][&type=问题类型][/code]

示例：
[code]curl "http://127.0.0.1:8000/api/query?title=中国的首都是哪里%3F&options=北京###上海###广州###深圳&type=选择题"[/code]

[b]2. 响应格式：[/b]
[code]{
  "code": 0,
  "msg": "获取成功",
  "data": {
    "data": "北京"
  }
}[/code]

[align=center][color=red][size=5][b]八、与OCS网课助手集成[/b][/size][/color][/align]

[b]1. 启动AI题库服务后，记录控制台输出的配置信息，例如：[/b]
[code]{
  "name": "完美题库",
  "homepage": "https://currso.com/",
  "url": "https://127.0.0.1:8000/api/query",
  "method": "get",
  "type": "GM_xmlhttpRequest",
  "contentType": "json",
  "data": {
    "title": "${title}",
    "options": "${options}",
    "type": "${type}"
  },
  "handler": "return (res)=>res.code === 0 ? [undefined, undefined] : [undefined,res.data.data]"
}[/code]

[b]2. 在OCS网课助手中添加题库配置：[/b]
[list]
[*]打开OCS网课助手
[*]进入题库设置
[*]添加新的题库配置，填入上述信息
[*]保存配置并启用
[/list]

[align=center][color=red][size=5][b]九、常见问题及解决方案[/b][/size][/color][/align]

[list]
[*][b]端口被占用：[/b]
  系统会自动检测端口冲突并切换到可用端口，查看控制台输出确认实际使用的端口号

[*][b]数据库连接失败：[/b]
  检查数据库配置是否正确，确认数据库服务是否启动，验证用户名和密码是否正确

[*][b]API调用失败：[/b]
  检查对应AI平台的API密钥是否正确配置，确认网络连接是否正常，查看控制台错误日志获取详细信息

[*][b]Windows脚本执行问题：[/b]
  使用PowerShell版本的配置工具，执行时添加 -ExecutionPolicy Bypass 参数
[/list]
